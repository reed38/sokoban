Fonction qui affiche les commandes disponibles en bas de terminal selon si le niveau est réussi et si le niveau suivant / précédent sont atteignables.

Paramètres
successBooléen : 1 si le niveau est réussi ; 0 sinon
reachablePreviousBooléen : 1 si le niveau précédent est atteingnable (ie si on n'est pas au niveau 1) ; 0 sinon
reachableNextBooléen : 1 si le niveau suivant est atteignable (s'il y a un niveau après ET si le niveau actuel est résolu) ; 0 sinon
